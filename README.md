### VSCode 사용자

1. eslint, prettier extension을 설치 (workspace 단위로 활성화 시킬것을 추천드립니다.)
2. Code > Preferences > Settings > Default formatter를 Prettier - Code formatter로 설정
3. 정상 적용 확인을 위하여 product-admin-client workspace vscode 재 실행
